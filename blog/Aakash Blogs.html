A new JSON-NOSQL Database on a Static Webpage

November 3, 2018

Getting a MongoDB type database on a static webpage that too in your HTML page. Well, How cool is that? Here's a thought....

MongoDB is a NoSQL database which unlike the traditional RDBMS databases stores data in JSON file format
whereas RDBMS databases stores data in tabular form. So, instead of actually pushing the data into the database
be it a NoSQL or RDBMS database, if the website or the webpage isin't a dyanmic one but static then one beautiful thing can be done.

WHICH IS: How about creating the JSON file ourselves and integrating it inside our static webpage and suppose if we need to find some
data out of that information onlythen we can perform a traverse, search and extract operation on the JSON data. This will be very cool thing to try on.

Blind Man see it all

November 10, 2018


Few years back no one would have imagined that there might come a time when blind people would also be able to see. Not seeing through actual eyes
No, but using the technology they will be able to know what exactly is happening in the world around them which till a couple of years back a science fiction
that has been converted into reality by Mr. Saqib Shaikh.
Sqib Shaikh who is a Microsoft developer, developed an Artificial Intelligence based eye-glass which is just like any other glass people wear but is a boon.

Let's know how and what it is.

Saqib Shaikh has developed a device which is capable of scanning everything in its surrounding and says it loud. Like it can detect people's emotion by the
look on their face or can tell if a girl is jumping off a skateboard or read you out the heading or the content of the menu of a very famous restraunt.
The most amazing thing is

SAQIB SHAIKH THE CREATOR OF THIS WONDERFUL TECHNOLOGY IS HIMSELF BLIND.
