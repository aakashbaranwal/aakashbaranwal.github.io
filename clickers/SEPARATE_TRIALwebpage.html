About Clickers

Clickers is an project initiative taken by the students of the engineering faculty: Aakash Baranwal, Ansh Agarwal and Apar Singhal.

This project is all about an interaction between a student and a teacher. So how does it works? : A teacher asks an Multiple Choice Question using the projector on the screen. Every student of the class will have his or her own remote which would have a Unique Identification Code to identify the owner of the remote. The student may click on any one of the four buttons which are actually the asnwer options marked as A, B, C and D.
The beauty of this project is that not only the answer response of the student is stored but also the attendence as well as a graph of the week's performance would be shown to the student so that they may assess themselves.
